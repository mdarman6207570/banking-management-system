# Banking Management System

A web-based Banking Management System developed using Spring Boot and MVC architecture.

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- JPA / Hibernate
- JSP
- MySQL
- i18n

## Features
- Customer Registration & Login
- Account Creation
- Deposit & Withdrawal
- Balance Inquiry
- Internationalization (i18n)

## Architecture
- Controller Layer
- Service Layer
- Repository Layer

## Database
- MySQL
- JPA/Hibernate ORM with automatic table generation

## How to Run
1. Clone the repository
2. Configure database details in `application.properties`
3. Run the project as Spring Boot Application
4. Access the application from browser
